# PropFlo Implementation Plan

## Purpose

This document defines the practical implementation plan after the proposal is approved.

The goal is to move from strategy and proposal into controlled execution.

## 1. Implementation Principle

Do not launch campaigns until the foundation is ready.

The correct order is:

1. Confirm compliance.
2. Confirm offer and claims.
3. Build landing pages.
4. Set up tracking.
5. Set up lead handling.
6. Prepare campaigns.
7. Test everything.
8. Launch with controlled budget.
9. Optimise weekly.

## 2. Phase 1: Proposal Approval

### Goal

Confirm that PropFlo accepts the proposed scope, budget, responsibilities, and launch approach.

### Tasks

- Present proposal
- Confirm once-off setup cost
- Confirm landing page cost
- Confirm monthly management cost
- Confirm suggested ad spend
- Confirm minimum term
- Confirm review point
- Confirm payment terms
- Confirm start date

### Output

- Signed proposal
- Approved budget
- Project start confirmed

## 3. Phase 2: Discovery And Compliance

### Goal

Confirm the product, legal, compliance, claims, and sales process before public marketing is created.

### Tasks

- Complete discovery meeting
- Complete compliance questions
- Confirm legal entity details
- Confirm product structure
- Confirm qualification rules
- Confirm fee and repayment wording
- Confirm required disclaimers
- Confirm privacy policy and terms
- Confirm approved claims
- Confirm restricted claims

### Output

- Completed compliance questions
- Approved claims list
- Restricted claims list
- Required disclaimer list

## 4. Phase 3: Access And Assets

### Goal

Collect the access and assets needed to build and track the marketing system.

### Tasks

- Collect logo files
- Collect brand colours
- Collect existing website access
- Collect domain or hosting access if needed
- Collect Google Analytics access
- Collect Google Tag Manager access
- Collect Google Ads access
- Collect Meta Business Manager access if needed
- Collect LinkedIn page access if needed
- Collect CRM or lead tracker access
- Collect privacy policy and terms links

### Output

- Access confirmed
- Assets confirmed
- Missing items listed

## 5. Phase 4: Landing Page Build

### Goal

Build the pages needed to convert paid and organic traffic into qualified enquiries.

### Priority Pages

1. Seller landing page
2. Check if you qualify form
3. Thank-you page
4. FAQ page

### Secondary Pages

1. How it works page
2. Estate agent partner page
3. Conveyancer partner page

### Tasks

- Write page copy
- Add approved disclaimers
- Add consent wording
- Add CTA buttons
- Add forms
- Add phone and WhatsApp links
- Add privacy policy links
- Test mobile layout
- Test page speed
- Get PropFlo approval

### Output

- Approved landing pages
- Approved forms
- Approved thank-you page

## 6. Phase 5: Tracking Setup

### Goal

Track enquiries, lead sources, and conversion actions before ads go live.

### Tasks

- Install or confirm Google Analytics 4
- Install or confirm Google Tag Manager
- Set up form submission tracking
- Set up thank-you page tracking
- Set up phone click tracking
- Set up WhatsApp click tracking
- Set up Google Ads conversion tracking
- Set up UTM structure
- Test lead source capture
- Test reporting

### Output

- Tracking tested
- Conversion events confirmed
- Test leads recorded

## 7. Phase 6: Lead Handling Setup

### Goal

Make sure PropFlo can respond to leads quickly and track outcomes.

### Tasks

- Confirm lead owner
- Confirm lead notification method
- Confirm response target
- Confirm lead stages
- Confirm qualification questions
- Confirm follow-up process
- Confirm WhatsApp scripts
- Confirm email templates
- Confirm weekly feedback process

### Output

- Lead handover process approved
- Lead stages approved
- Follow-up workflow approved

## 8. Phase 7: Campaign Setup

### Goal

Prepare campaigns for controlled launch.

### Tasks

- Build Google Search campaign
- Add keyword groups
- Add negative keywords
- Add ad copy
- Add location targeting
- Add conversion actions
- Set initial daily budget
- Prepare Meta retargeting campaign if approved
- Prepare LinkedIn partner activity if approved

### Output

- Campaigns built
- Campaigns reviewed
- Campaigns ready for launch

## 9. Phase 8: Pre-Launch Testing

### Goal

Catch technical, tracking, wording, and lead routing problems before spending money.

### Test

- Landing page loads correctly
- Mobile display works
- Forms submit correctly
- Thank-you page works
- Email notification is received
- WhatsApp links work
- Phone links work
- GA4 event fires
- Google Ads conversion fires
- UTM data is captured
- Lead appears in tracker or CRM
- Disclaimer is visible
- Privacy policy link works

### Output

- Launch checklist approved

## 10. Phase 9: Controlled Launch

### Goal

Launch with limited budget and monitor closely.

### Tasks

- Launch Google Search campaign
- Check spend daily at first
- Check search terms
- Check conversions
- Check lead notifications
- Check lead quality
- Add negative keywords
- Pause poor terms
- Record optimisation notes

### Output

- Week 1 launch review
- First optimisation actions

## 11. Phase 10: Weekly Optimisation

### Goal

Improve lead quality and reduce wasted spend.

### Weekly Tasks

- Review Google Ads
- Review search terms
- Review cost per lead
- Review qualified leads
- Review not-qualified reasons
- Review landing page questions
- Review sales feedback
- Update optimisation log

### Output

- Weekly optimisation notes
- Recommended changes

## 12. Phase 11: Monthly Reporting

### Goal

Report on performance and decide what to change next.

### Monthly Report Includes

- Spend
- Leads
- Qualified leads
- Applications
- Approvals
- Funded deals
- Cost per lead
- Cost per qualified lead
- Cost per funded deal
- Campaign changes
- Issues
- Recommendations

### Output

- Monthly report
- Next-month recommendations
- Budget recommendation

## 13. Phase 12: 90-Day Review

### Goal

Decide whether to scale, adjust, or restructure.

### Review Questions

1. Which channels produced qualified leads?
2. What was the cost per qualified lead?
3. What was the cost per funded deal?
4. Did PropFlo handle leads quickly enough?
5. Did the landing pages explain the offer clearly?
6. Which keywords should be expanded?
7. Which channels should be reduced?
8. Should budget increase?
9. Should partner marketing become a bigger focus?
10. What should happen next?

### Output

- 90-day review
- Scaling recommendation
- Updated strategy

## 14. Starting Recommendation

Use this implementation plan only after PropFlo approves the proposal and answers the compliance questions.

Until then, the project is still in preparation stage.
