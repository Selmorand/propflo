# PropFlo Marketing Project Index

## Purpose

This is the main index for the PropFlo marketing project.

Use this file to understand what each folder contains and where to find the working documents.

## 1. Project Goal

Build a professional marketing operating system for PropFlo.

The system should support:

- Strategy
- Proposal preparation
- Budget planning
- Compliance preparation
- Landing page planning
- Paid advertising
- SEO content
- Partner marketing
- Lead tracking
- Follow-up workflows
- Monthly reporting
- Ongoing optimisation

## 3. Recommended Working Order

Use this order:

1. Complete compliance questions.
2. Confirm product details with PropFlo.
3. Finalise approved claims and disclaimers.
4. Finalise strategy.
5. Finalise scope and budget.
6. Present proposal.
7. Get approval and deposit.
8. Build landing pages.
9. Set up tracking.
10. Prepare campaigns.
11. Test forms and conversions.
12. Launch controlled campaign.
13. Review weekly.
14. Report monthly.
15. Optimise and scale.

## 4. Important Rule

Do not launch advertising until these are complete:

- Compliance questions answered
- Approved claims confirmed
- Disclaimers confirmed
- Landing page approved
- Tracking tested
- Lead handover agreed
- Client approval received
