# PropFlo Approval Log

## Purpose

This document records approvals for PropFlo marketing material.

Because PropFlo operates in a finance-related space, approvals must be tracked before public use.

Use this file to record who approved what, when it was approved, and whether any conditions were attached.

## 1. Approval Principle

No public-facing material should go live until the relevant person at PropFlo has approved it.

This includes:

- Website copy
- Landing page copy
- Ad copy
- Google Ads headlines and descriptions
- Meta ad copy
- LinkedIn posts
- Email templates
- WhatsApp scripts
- Partner material
- Brochures
- Disclaimers
- Qualification wording
- Fee wording
- Repayment wording
- Approval or payout timing claims

## 3. Approval Methods

Use one of these:

- Email approval
- WhatsApp approval
- Meeting approval
- Signed document
- Comment approval
- Platform approval

## 4. Version Control

Each item should have a version number.

Example:

- Seller landing page v1
- Google Ads copy v1
- WhatsApp scripts v1
- Partner PDF v1
- FAQ page v2

If wording changes later, get new approval and record it as a new version.

## 6. Approval Notes

Record any conditions.

Examples:

- Approved only if disclaimer is shown below the form.
- Approved only for Google Search Ads, not Meta.
- Approved only after legal review.
- Approved but fee wording must be removed.
- Approved for partner use only.
