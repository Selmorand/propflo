# PropFlo Monthly Report Data Checklist

## Purpose

This document lists the data needed before preparing the monthly PropFlo marketing report.

The goal is to avoid writing reports based only on ad clicks or surface-level numbers.

## 1. Main Principle

A useful monthly report must connect marketing activity to business outcomes.

Do not report only on:

- Impressions
- Clicks
- Likes
- Reach
- Website visits

Also report on:

- Qualified leads
- Applications
- Approvals
- Funded deals
- Cost per qualified lead
- Cost per funded deal
- Lead response time
- Reasons leads did not convert

## 3. Data Needed From Advertising Platforms

Collect:

- Google Ads spend
- Google Ads impressions
- Google Ads clicks
- Google Ads conversions
- Google Ads cost per conversion
- Google Ads search terms
- Google Ads keyword performance
- Google Ads campaign performance
- Google Ads ad group performance
- Meta spend, if used
- Meta reach, if used
- Meta clicks, if used
- Meta conversions, if used
- LinkedIn performance, if used

## 4. Data Needed From Website Analytics

Collect:

- Website sessions
- Users
- New users
- Landing page visits
- Seller landing page performance
- FAQ page visits
- How it works page visits
- Partner page visits
- Traffic source and medium
- Device performance
- Form starts, if tracked
- Form submissions
- Phone clicks
- WhatsApp clicks
- Email clicks

## 5. Data Needed From Lead Tracker Or CRM

Collect:

- Total leads
- Seller leads
- Partner leads
- Qualified leads
- Not-qualified leads
- Duplicate leads
- Spam leads
- Leads contacted
- Leads not contacted
- Documents requested
- Documents received
- Applications submitted
- Applications approved
- Applications declined
- Offers sent
- Offers accepted
- Funded deals
- Lost leads

## 6. Data Needed From PropFlo Sales Team

Ask PropFlo for:

- Which leads were good?
- Which leads were poor?
- Which leads qualified?
- Which leads did not qualify?
- Why did leads not qualify?
- Which leads became applications?
- Which leads were approved?
- Which leads were funded?
- Which leads were lost?
- Why were deals lost?
- How quickly were leads contacted?
- What questions did leads ask repeatedly?
- What objections appeared?
- Were there process delays?

## 7. Financial Performance Data

Collect if PropFlo is willing to share:

- Average funded deal value
- Revenue per funded deal
- Gross profit per funded deal
- Total funded value
- Total revenue from marketing leads
- Estimated return on marketing spend

If PropFlo does not share this data, report only up to cost per qualified lead or cost per application.

## 9. Content And SEO Data

## 10. Tracking Issues

Record:

- Broken tracking
- Missing UTM data
- Untracked phone calls
- Untracked WhatsApp leads
- CRM update gaps
- Offline conversion gaps
- Form issues
- Platform reporting delays

## 11. Report Preparation Checklist

Before preparing the report, confirm:

- Ad spend is correct
- Lead count is correct
- Qualified lead count is correct
- Not-qualified reasons are available
- Funded deal data is available or marked unavailable
- Tracking gaps are noted
- Optimisation changes are recorded
- Budget recommendation is prepared
- Next-month action list is prepared

## 12. Monthly Report Summary Questions

Answer these before writing the report:

1. What was the main result this month?
2. What worked?
3. What did not work?
4. What caused wasted spend?
5. What improved lead quality?
6. What should be paused?
7. What should be expanded?
8. What must PropFlo fix internally?
9. What should Interon improve next month?
10. What budget is recommended next month?

## 13. Starting Recommendation

Collect sales feedback before writing the report.

Without PropFlo sales feedback, the report can only show marketing activity, not true business performance.
