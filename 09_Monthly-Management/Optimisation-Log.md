# PropFlo Optimisation Log

## Purpose

This document records campaign, website, content, tracking, and lead-quality changes made during monthly management.

The goal is to keep a clear history of what was changed, why it was changed, and what happened afterwards.

## 1. Main Principle

Every optimisation should have a reason.

Do not change campaigns randomly.

Each change should be linked to one of the following:

- Improve lead quality
- Reduce wasted spend
- Improve conversion rate
- Fix tracking
- Improve landing page clarity
- Improve sales handover
- Improve partner response
- Improve reporting accuracy

## 2. Optimisation Log Format

Use this format for each change:

### Date

YYYY-MM-DD

### Area

Examples:

- Google Ads
- Landing page
- Tracking
- SEO content
- Partner marketing
- Lead handling
- WhatsApp follow-up
- Email follow-up

### Issue

What problem or opportunity was identified?

### Change Made

What was changed?

### Reason

Why was the change made?

### Expected Result

What should improve?

### Follow-Up Date

When should this be checked again?

### Result

What happened after the change?

## 3. Example Entry

### Date

2026-07-03

### Area

Google Ads

### Issue

Search terms showed several irrelevant personal loan searches.

### Change Made

Added negative keywords for payday loan, personal loan, instant loan, and bad credit loan.

### Reason

These searches do not match PropFlo's target audience of property sellers.

### Expected Result

Reduced wasted spend and improved lead quality.

### Follow-Up Date

2026-07-10

### Result

To be reviewed.

## 4. Google Ads Optimisation Items

Record changes such as:

- Negative keywords added
- Keywords paused
- Keywords added
- Match types adjusted
- Ad copy changed
- Budget changed
- Campaign paused
- Ad group paused
- Location targeting changed
- Device adjustment made
- Conversion tracking fixed

## 5. Landing Page Optimisation Items

Record changes such as:

- CTA wording changed
- FAQ added
- Form field added
- Form field removed
- Disclaimer updated
- Trust section added
- Page section reordered
- Mobile layout fixed
- Thank-you page changed

## 6. Lead Quality Optimisation Items

Record changes such as:

- Form qualification question added
- Poor lead reason identified
- Lead stage updated
- Sales feedback added
- Referral source reviewed
- Response time issue noted
- Not-qualified pattern identified

## 7. Content Optimisation Items

Record changes such as:

- Blog topic added
- FAQ updated
- Internal link added
- Existing page improved
- Search query used for content idea
- Partner question turned into content

## 8. Partner Marketing Optimisation Items

Record changes such as:

- Partner message updated
- Partner PDF changed
- Follow-up sequence changed
- Partner FAQ updated
- Referral process clarified
- Partner source tracked

## 9. Monthly Summary

At the end of each month, summarise:

- Main changes made
- Changes that improved results
- Changes that did not work
- New risks found
- Next changes recommended

## 10. Starting Recommendation

Update this log every time a meaningful change is made.

This becomes useful evidence in monthly reporting and protects against vague client conversations.
